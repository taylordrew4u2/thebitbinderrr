# BitBinder Project Overview

## Purpose
BitBinder is a **professional comedy notebook app** designed by comedians for comedians. It serves as a comprehensive digital tool for comedy writers and performers.

### Core Features
- **📝 Digital Notepad**: Quick joke writing with ruled lines and authentic notebook feel
- **🎭 Joke Management**: Organize jokes into folders and categories with AI-powered categorization
- **📋 Set Lists**: Create and manage comedy set lists for performances  
- **🎙️ Audio Recording**: Record performances with auto-transcription capabilities
- **📸 Notebook Scanner**: Scan physical notebooks for digital backup using OCR
- **🔥 Roast Mode**: Specialized writing mode for targeted comedy and roast content
- **☁️ iCloud Sync**: Full cross-device synchronization with robust data protection
- **🤖 BitBuddy AI**: AI assistant for comedy writing help and organization

## Target Users
- Stand-up comedians
- Comedy writers  
- Performers who want to organize and develop their material
- Anyone writing jokes or humorous content professionally

## App Philosophy
- **Data is sacred**: Never lose user's comedy material
- **Real notebook feel**: Authentic writing experience with visual cues (holes, lines, margins)
- **Professional tools**: Features that support actual comedy workflow
- **Always accessible**: Works offline with cloud backup when available
